# Bionic Reading Converter

A simple, clean web-based tool that converts regular text into **bionic reading format** to improve reading speed and reduce cognitive fatigue.

## What is Bionic Reading?

Bionic reading is a technique that highlights the first letters of each word, guiding your eyes and helping your brain complete words faster. This reduces:
- Eye movement and strain
- Cognitive load during reading
- Reading fatigue
- Time spent re-reading passages

## How to Use

1. Open `index.html` in your browser
2. Paste or type your text in the input area
3. Adjust the bold ratio slider (default: 50%)
4. Click "Convert to Bionic"
5. Copy the result or read directly

**Keyboard shortcut:** Press `Ctrl+Enter` in the text area to convert

## Live Demo

Simply download the repository and open `index.html` in any modern browser. No build process or server required!

## Why This Helps

If you struggle with:
- Losing focus while reading

Bionic reading can make text processing significantly easier by reducing the cognitive effort required.

## Technical Details

- Pure vanilla JavaScript (no frameworks)
- Responsive CSS Grid/Flexbox layout
- Calculates bold length dynamically based on word length
- Works offline once loaded

## Customization

You can easily modify:
- **Bold ratio:** Adjust the slider range in `index.html` (lines with `min`, `max`, `value`)
- **Colors:** Change the gradient and accent colors in the CSS
- **Font:** Modify the `font-family` in the CSS

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
